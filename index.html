🔧 What is GhostX?

GhostX is a stealthy, modular hacking toolkit for web reconnaissance, vulnerability detection, and deep scraping. Designed for both ethical hackers and researchers, GhostX provides an arsenal of powerful commands — all cloaked under the radar.

🧩 Key Features

  • 🕵️‍♂️ Social Recon (GitHub, Twitter, etc.)
  • 🌐 Advanced Web Scraping Engine
  • 🛡️ CSRF & XSS Vulnerability Scanners
  • 📡 Port Scanner & Reverse DNS Lookup
  • 💀 Discreet User-Agent Rotation
  • 👻 Ghost Mode: no logs, no traces

📟 Core Commands

  • ghostx social github [username] — GitHub profile recon
  • ghostx search link [domain] — Extract all links from a site
  • ghostx scan csrf [url] — Scan for CSRF vulnerabilities
  • ghostx portscan [host] — Lightweight stealth port scan
  • ghostx utils useragents — View/disguise User-Agents

🚀 Getting Started

Clone the repo and execute in terminal:

git clone https://github.com/vihaanmody1/ghostx  
cd ghostx  
python ghostx.py

Make sure you have Python 3.10+ and required modules installed.
